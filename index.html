- Every number is randomly generated between 1-10.
- Every node is connected to the four nodes on its north, east, south, west direction.
- The higher the number is the higher the cost which the path avoids

-Start: Begin at the chosen starting node.
-Explore Neighbors:
Check each neighbor of the current node.
Calculate the cost to reach the neighbor (current node's cost + neighbor's difficulty)
If this cost is lower, update the neighbor's cost and mark the current node as its parent.
-Choose Next: Select the unvisited node with the lowest cost.
-Repeat: Continue exploring neighbors and choosing the next node until the end is reached or no more nodes are reachable.
